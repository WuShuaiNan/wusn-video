# wusn-video
- `达娃里氏`个人视频监控微服务。

## 项目简介
- 本项目主要用于视频协议之间的转换，并封装为 dubbo 接口，供 wusn-api 接口集成服务使用， 并发布给 wusn-ui 前端项目。
- 已实现的协议转换如下：
  - rtsp -> flv
